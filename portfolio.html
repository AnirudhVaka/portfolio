CI/CD ARCHITECTURE

AICPA Actions Platform

Spearheaded the organization's DevOps migration, replacing an error-prone sequential model (develop → sit) with a deeply automated, Label-Driven GitOps Topology (Main → Release). Completely eliminated manual cherry-picking and prod deployments.

  • Full-Stack FBEs: Feature PRs dynamically provision complete AWS sets (ECS, RDS, S3, SQS) via an `fbe` label for exhaustive isolated testing.
  • Teams Deployment Ops: Integrated Power Automate hooks into the CD pipeline for real-time global deployment visibility in MS Teams.
  • Zero-Touch Prod Releases: Merging to `main` fires Prod deployments with automated ServiceNow CR creation and keyless OIDC authentication.
  • ServiceNow ITSM: The production pipeline programmatically creates a Change Request before initialization and closes it upon successful E2E validation.
